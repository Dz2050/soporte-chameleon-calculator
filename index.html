Kameleon Calculator – Política de Privacidad

Nombre de la aplicación: Kameleon Calculator
Desarrollador: David Aguilar de Jesus
Última actualización: Marzo 2026

1. Información recopilada

Kameleon Calculator no recopila, almacena, transmite ni comparte información personal de los usuarios. La aplicación no requiere registro ni creación de cuentas.

2. Permisos utilizados

Acceso a imágenes y archivos del dispositivo:

La aplicación solicita acceso al almacenamiento únicamente para permitir que el usuario seleccione una imagen y personalice el fondo de la calculadora.

Las imágenes seleccionadas permanecen almacenadas localmente en el dispositivo del usuario y no se cargan, transmiten ni comparten con ningún servidor externo.

3. Datos que NO se recopilan

  • No accede a la cámara
  • No graba audio
  • No recopila información personal
  • No solicita cuentas de usuario
  • No registra historial de uso
  • No utiliza servicios de análisis (analytics)
  • No comparte datos con terceros

4. Servicios de terceros

Kameleon Calculator no utiliza servicios de terceros que recopilen datos personales.

5. Almacenamiento y seguridad

Todas las configuraciones y personalizaciones realizadas por el usuario se almacenan exclusivamente en el dispositivo del usuario. No se realiza ninguna transmisión de datos a servidores externos.

6. Derechos del usuario

Dado que la aplicación no recopila datos personales, no se almacenan ni procesan datos del usuario por parte del desarrollador.
